Quantifai Sync

This installer will set up quantifai-sync, a lightweight background agent that streams your Claude Code usage data to your Quantifai dashboard.

After installation, the agent runs automatically in the background. It watches your Claude Code session files and syncs usage metrics to your configured API endpoint.

What gets installed:

  • The quantifai-sync binary in /usr/local/bin/
  • A LaunchAgent for automatic startup at login
